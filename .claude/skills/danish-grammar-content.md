# Skill: Danish Language Content & Grammar

## Danish Exam Structure Reference

### Module Tests (Modultests)
- Module 1: Very basic Danish (A1)
- **Module 2**: Elementary Danish (A2) — **MVP target**
- Module 3: Pre-intermediate (A2+)
- Module 4: Intermediate (B1)
- Module 5: Upper-intermediate (B2) → leads to PD3

### Prøve i Dansk (PD) Exams
- PD1: A1 written / A2 oral — end of DU1 pathway
- PD2: A2/B1 — end of DU2 pathway
- PD3: B2 — end of DU3 pathway (Module 5). Required for citizenship.
- Studieprøven: C1 — Module 6. Required for university.

### PD3 Exam Format
- **Reading 1**: 25 min — find factual info in texts
- **Reading 2**: 65 min — 3 parts: comprehension, paragraph ordering, fill-in words
- **Written**: 2 parts — letter/email + essay
- **Oral**: 2 parts — prepared topic presentation + spontaneous discussion

## Module 2 Grammar — Detailed Content

### 1. T-ord og N-ord (Noun Gender)

**Rules to teach:**
- En-words (fælleskøn/common gender): ~75% of Danish nouns
- Et-words (intetkøn/neuter gender): ~25% of Danish nouns
- No reliable rule to determine gender — must be memorized
- Some patterns help:
  - People/animals are usually en-words: en mand, en kvinde, en hund
  - Abstract nouns ending in -hed, -else, -tion, -itet are en-words
  - Countries and languages are et-words: et land, et sprog
  - Collective nouns often et-words: et folk, et parti

**Forms to practice:**

| Form | En-word | Et-word |
|------|---------|---------|
| Indefinite singular | en bil | et hus |
| Definite singular | bilen | huset |
| Indefinite plural | biler | huse |
| Definite plural | bilerne | husene |

**Irregular plural patterns:**
- No change: et barn → børn (irregular)
- Vowel change: en mand → mænd
- -er ending: en ting → ting (zero plural)

### 2. Komparativ og Superlativ

**Regular (-ere / -est):**
```
stor → større → størst
lang → længere → længst
smuk → smukkere → smukkest
```

**With mere/mest (multi-syllable adjectives):**
```
interessant → mere interessant → mest interessant
praktisk → mere praktisk → mest praktisk
```

**Irregular (must memorize):**
```
god → bedre → bedst
dårlig → værre → værst
lille → mindre → mindst
stor → større → størst
gammel → ældre → ældst
ung → yngre → yngst
mange → flere → flest
meget → mere → mest
```

**Agreement:**
```
Den store bil (the big car) — definite, uses e-form
Et stort hus (a big house) — neuter indefinite, uses t-form
De store huse (the big houses) — plural, uses e-form
```

### 3. Omvendt Ordstilling (Inverted Word Order)

**V2 Rule:** The verb is ALWAYS in second position in main clauses.

**Normal order:** Subject – Verb – Object
```
Jeg spiser morgenmad.
(I eat breakfast.)
```

**Inverted order (when something else is in first position):**

Time expression first:
```
I morges spiste jeg morgenmad.
(This morning I ate breakfast.) — verb before subject
```

Place first:
```
I Danmark taler man dansk.
(In Denmark people speak Danish.)
```

Question:
```
Hvad laver du?
(What are you doing?)
```

Subordinate clause first:
```
Hvis det regner, tager jeg en paraply.
(If it rains, I take an umbrella.)
```

**Key rule:** After subordinate clause at start, main clause inverts: conjunction-clause, VERB SUBJECT...

### 4. Hovedsætning og Ledsætninger

**Main clause (Hovedsætning) — SVAO:**
```
Subject – Verb – Adverb – Object
Han spiser ikke morgenmad.
(He doesn't eat breakfast.)
```

**Subordinate clause (Ledsætning) — CSAVO:**
```
Conjunction – Subject – Adverb – Verb – Object
...fordi han ikke spiser morgenmad.
(...because he doesn't eat breakfast.)
```

**Critical difference:** Adverb (ikke, altid, aldrig, ofte, etc.) goes AFTER verb in main clause, BEFORE verb in subordinate clause.

**Common subordinating conjunctions:**
- at (that)
- fordi (because)
- hvis (if)
- når (when — repeated/future)
- da (when — past, single event)
- selvom (even though)
- mens (while)
- inden (before)
- efter at (after)
- så (so)
- medmindre (unless)

**Common coordinating conjunctions (no inversion):**
- og (and)
- men (but)
- eller (or)
- for (because — formal)
- så (so — when starting a new main clause)

### 5. Verber og Tider

**Regular verb groups:**

Group 1 (-ede verbs):
```
at snakke → snakker → snakkede → har snakket
at arbejde → arbejder → arbejdede → har arbejdet
```

Group 2 (-te verbs):
```
at spise → spiser → spiste → har spist
at læse → læser → læste → har læst
at købe → køber → købte → har købt
```

**Irregular verbs (must memorize):**
```
at gå → går → gik → har gået
at komme → kommer → kom → har kommet (er kommet)
at se → ser → så → har set
at tage → tager → tog → har taget
at give → giver → gav → har givet
at drikke → drikker → drak → har drukket
at skrive → skriver → skrev → har skrevet
at finde → finder → fandt → har fundet
at ligge → ligger → lå → har ligget
at sige → siger → sagde → har sagt
```

**Tense usage:**
- Nutid (present): habitual, ongoing, scheduled future
- Datid (past): completed past events, narratives
- Førnutid (present perfect): past with present relevance, experiences
- Førdatid (past perfect): event before another past event
- Fremtid (future): vil/skal + infinitive

### 6. Pronominer

**Personal pronouns:**

| Person | Subject | Object |
|--------|---------|--------|
| 1sg | jeg | mig |
| 2sg | du | dig |
| 3sg m | han | ham |
| 3sg f | hun | hende |
| 3sg (en) | den | den |
| 3sg (et) | det | det |
| 1pl | vi | os |
| 2pl | I | jer |
| 3pl | de | dem |

**Possessive pronouns:**

| Person | En-word | Et-word | Plural |
|--------|---------|---------|--------|
| 1sg | min | mit | mine |
| 2sg | din | dit | dine |
| 3sg (own) | sin | sit | sine |
| 3sg m (not own) | hans | hans | hans |
| 3sg f (not own) | hendes | hendes | hendes |
| 1pl | vores | vores | vores |
| 2pl | jeres | jeres | jeres |
| 3pl | deres | deres | deres |

**sin/sit/sine vs hans/hendes:**
```
Peter vasker sin bil. (Peter washes his [own] car.)
Peter vasker hans bil. (Peter washes his [someone else's] car.)
```

**Reflexive pronouns:** mig, dig, sig, os, jer, sig
```
Jeg vasker mig. (I wash myself.)
Han barberer sig. (He shaves himself.)
```

## Common Mistakes to Test

These are what exam takers frequently get wrong — prioritize exercises testing these:

1. Wrong noun gender (en/et confusion)
2. Adverb placement in subordinate clauses (ikke BEFORE verb)
3. Forgetting inversion after initial adverb/clause
4. Mixing up sin/hans/hendes
5. Wrong past tense for irregular verbs
6. Using mere/mest with short adjectives that take -ere/-est
7. Forgetting the -t/-e agreement on adjectives
8. Confusing når/da for "when"
9. Wrong word order in questions
10. Using the wrong pronoun case (subject vs object)
